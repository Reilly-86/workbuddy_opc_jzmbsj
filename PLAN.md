# 建筑模版商机系统 - 详细执行计划

## 一、项目目标

为建材模板/木方厂家构建**上海在建工地商机自动搜集系统**：
- 每天早上 8:00 自动运行
- 搜集上海地区在建工地完整信息
- 增量对比，标注每日新增项目
- 输出带筛选/搜索功能的 HTML 报告
- 核心目的：**找到项目 → 找到承建方 → 联系承建方 → 卖模板**

---

## 二、数据字段定义

### 基础信息
| 字段 | 说明 | 来源 |
|------|------|------|
| 项目名称 | 工程全称 | 住建委/交易中心 |
| 所属区域 | 如浦东、嘉定、青浦等 | 住建委/审批平台 |
| 项目实际地址 | 精确到道路/路口级别 | 发改委/审批公示 |
| 项目类别 | 科技产业/交通枢纽/轨道交通/医疗/教育/市政/住宅 | 自动分类 |
| 开工时间 | 实际或计划开工日期 | 施工许可证 |
| 预计完工时间 | 计划竣工日期 | 施工许可证 |

### 投资与体量
| 字段 | 说明 | 来源 |
|------|------|------|
| 项目总投资额 | 万元 | 审批公示 |
| 建筑面积 | ㎡ | 施工许可证 |
| 模板需求预估 | 粗算：建筑面积 × 经验系数 | 自动计算 |
| BD优先级 | ⭐~⭐⭐⭐ 综合评分 | 自动计算 |

### 承建方信息
| 字段 | 说明 | 来源 |
|------|------|------|
| 承建单位全称 | 施工总承包单位名称 | 中标公示/住建委 |
| 承建单位官网 | 官网地址 | 企查查/手动验证 |
| 统一社会信用代码 | 用于进一步查询 | 住建委/中标公示 |
| 资质等级 | 特级/一级/二级等 | 住建委企业库 |
| 联系电话 | 总机或采购部电话 | 企查查/天眼查 |
| 采购负责人（如有） | 姓名+职务 | 企查查/招投标公告 |
| 地址 | 承建方注册/办公地址 | 企查查 |

### 招投标信息
| 字段 | 说明 | 来源 |
|------|------|------|
| 招标公示网站 | 发布招标的网站 | 招标公告 |
| 中标公示链接 | 直接链接到中标结果 | 交易中心 |
| 中标金额 | 万元 | 中标公示 |
| 中标时间 | 中标公告发布日期 | 中标公示 |

### 运营字段
| 字段 | 说明 |
|------|------|
| 是否新增 | 与昨日数据对比，新增=🆕 |
| 首次发现日期 | 第一次入库的日期 |
| 最近更新日期 | 信息最后更新时间 |

## 三、数据筛选规则

### 纳入标准（同时满足）
1. 项目地点在上海市范围内
2. 项目状态为"在建"或"新开工"
3. 建筑面积 ≥ 5,000㎡（或投资额 ≥ 3000 万元）
4. 类型覆盖：住宅（大型）、商业、工业/厂房、公共建筑（医院/学校/交通）、市政工程

### 排除标准（任一即排除）
1. 建筑面积 < 5,000㎡ 的小型项目
2. 投资额 < 3000 万元的零星工程
3. 纯装修/改造/修缮项目（不涉及主体结构模板）
4. 已竣工项目
5. 临时设施/围挡工程

## 五、执行步骤（分三期）

### Phase 1：手动验证 + 数据源打通（本次执行）
> 目标：验证数据源可行性，手动完成一期 HTML 报告

| 步骤 | 内容 | 产出 |
|------|------|------|
| 1.1 | 从上海住建委/交易中心抓取在建项目列表（施工许可） | 原始项目数据 |
| 1.2 | 从中标公示补全承建单位信息 | 承建方数据 |
| 1.3 | 用企查查/天眼查补全承建方联系方式（尽力获取） | 联系方式数据 |
| 1.4 | 数据清洗、去重、筛选 | 清洗后数据集 |
| 1.5 | 计算 BD 优先级和模板需求预估 | 评分结果 |
| 1.6 | 生成完整 HTML 报告（含全部字段 + 筛选搜索） | `report.html` |

### Phase 2：自动化 Pipeline
> 目标：每天 8:00 自动运行

| 步骤 | 内容 |
|------|------|
| 2.1 | 编写 Python 数据采集脚本 |
| 2.2 | 实现增量对比逻辑（JSON 存储 + 对比） |
| 2.3 | HTML 模板化生成（Jinja2） |
| 2.4 | 配置 WorkBuddy Automation 定时任务 |

### Phase 3：全国扩展（后续）
> 目标：从上海扩展到全国重点城市

| 步骤 | 内容 |
|------|------|
| 3.1 | 接入全国建筑市场监管平台 |
| 3.2 | 按省份/城市分页采集 |
| 3.3 | 添加区域筛选功能 |

## 六、BD优先级评分规则

综合评分 = 体量分(40%) + 时效分(30%) + 承建方分(30%)

| 维度 | 评分标准 |
|------|---------|
| **体量分** | 面积>5万㎡=10分, 2-5万㎡=7分, 0.5-2万㎡=4分 |
| **时效分** | 开工<3个月=10分, 3-6个月=7分, 6-12个月=4分, >12个月=2分 |
| **承建方分** | 特级资质=10分, 一级=7分, 二级=4分, 其他=2分 |

| 总分 | 优先级 | 建议 |
|------|-------|------|
| 8-10 | ⭐⭐⭐ | 立即跟进，实地拜访 |
| 5-7 | ⭐⭐ | 本周电话联系 |
| 3-4 | ⭐ | 纳入观察名单 |

## 七、模板需求估算参考

| 建筑类型 | 经验系数（模板面积/建筑面积） | 说明 |
|---------|---------------------------|------|
| 住宅（高层） | 2.0-2.5 | 标准层重复率高 |
| 商业/办公 | 1.5-2.0 | 结构多样 |
| 工业厂房 | 0.8-1.2 | 单层为主 |
| 医院/学校 | 1.8-2.3 | 结构较复杂 |
| 市政工程 | 按项目另算 | 桥梁/隧道差异大 |
